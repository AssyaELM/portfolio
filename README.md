# Portfolio — Assia El Mourtadah

Portfolio personnel d'**Assia El Mourtadah**, étudiante ingénieure en Développement Numérique et Cybersécurité à l'ENSA Fès.

🌐 **Site en ligne : [portfolio-assia.netlify.app](https://portfolio-assia.netlify.app)**

## Aperçu

Portfolio interactif au thème cyberpunk :
- Intro animée façon terminal (`whoami --detail`)
- Fond néon animé réactif à la souris (canvas)
- Profil, compétences, 8 projets cybersécurité avec liens GitHub
- Certifications avec images (lightbox), expériences et activités
- 100 % HTML/CSS/JS vanilla — aucun framework, aucune dépendance

## Structure

```
index.html        # Le portfolio complet (CSS et JS inline)
assets/           # Photo de profil et images des certificats
```

## Déploiement

Site statique hébergé sur **Netlify**. Pour mettre à jour : pousser sur `main` puis redéployer le dossier sur Netlify.
